4DInference

Novel 4D inference architecture with learned routing.

This repository contains Julia code and experiments for the 4DInference project (slices, router, memory, phase, etc.). See AGENTS.md in the workspace for project-level details and run commands.

Quick start

- To run tests: `cd 4DInference && julia --project=. test/runtests.jl`
- Examples: `cd 4DInference && julia --project=. examples/sequence_experiment.jl`
